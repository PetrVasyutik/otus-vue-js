# E2E Тесты

Этот каталог содержит end-to-end тесты, написанные с использованием Playwright.

## Установка

Перед запуском тестов необходимо установить зависимости:

```bash
npm install
```

После установки зависимостей нужно установить браузеры для Playwright:

```bash
npx playwright install
```

## Запуск тестов

### Запуск всех E2E тестов

```bash
npm run test:e2e
```

### Запуск с UI режимом (интерактивный)

```bash
npm run test:e2e:ui
```

### Запуск в headed режиме (с видимым браузером)

```bash
npm run test:e2e:headed
```

### Запуск конкретного теста

```bash
npx playwright test e2e/login.spec.js
```

## Структура тестов

- `add-to-cart.spec.js` - Тесты для добавления товаров в корзину
- `login.spec.js` - Тесты для проверки логина
- `load-data.spec.js` - Тесты для загрузки данных
- `cart-calculation.spec.js` - Тесты для расчета суммы в корзине

## Примечания

- Тесты автоматически запускают dev сервер перед выполнением
- Базовый URL: `http://localhost:5173`
- Убедитесь, что порт 5173 свободен перед запуском тестов
